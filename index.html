Meu nome é Milena Ramalho, tenho 21 anos e sou de Santos. Adoro jogos, livros e séries de ficção e tenho um hiperfoco em salsichinhas (os doguinhos compridinhos).

Desde bem novinha possuo uma paixão genuína por tecnologia e criação de conteúdos gráficos, e hoje exercito essa paixão todos os dias na minha área de estudos: o desenvolvimento web e front-end.

Atualmente estou no primeiro período do Tecnólogo de Análise e Desenvolvimento de Sistemas pela Universidade Católica de Santos.

Meus Projetos

Calculadora de IMC

O projeto "Calculadora de IMC" é uma aplicação web interativa desenvolvida usando HTML, CSS e JavaScript cujo objetivo principal é fornecer uma ferramenta fácil de usar para calcular o Índice de Massa Corporal (IMC) e interpretar os resultados de forma clara.

Contagem Regressiva

O projeto 'Contagem Regressiva' desenvolvido com HTML, CSS e JavaScript é apresentado em uma interface limpa e intuitiva, com objetivo de informar o tempo restante até o evento em dias, horas, minutos e segundos.
